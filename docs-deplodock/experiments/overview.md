# Experiments

Every benchmark run is automatically saved as an experiment — a snapshot of the configuration,
GPU type, and results.

## What's tracked

- Model and configuration used
- GPU type and count
- Full benchmark results (TTFT, TPS, ITL, memory)
- Timestamp and run duration

## Sharing

You can make any experiment public with a shareable link. Public experiments appear
in the community leaderboard, where you can compare results across users, models, and GPUs.
