# Custom Recipes

Custom recipes allow users and teams to create reusable configurations for virtual machines and containers. Instead of specifying instance parameters each time, you can define a recipe once and use it to quickly launch instances with your preferred settings.

## Overview

- **User recipes** — Personal recipes tied to your account.
- **Team recipes** — Shared recipes available to all members of a team.
- **VM and Docker support** — Recipes can be created for both virtual machine and Docker container instances.

## Managing Recipes

Recipes can be managed through the CloudRift REST API. See the [API documentation](https://api.cloudrift.ai/swagger-ui/) for the full list of recipe endpoints.
