# Instance Reservations

Reservations allow you to lock in lower pricing for your CloudRift instances. Instead of paying on-demand rates, you can convert running instances into reserved ones and save on costs.

## How Reservations Work

1. **Rent an instance** as usual through the CloudRift Console or API.
2. **Convert to reserved** — Once your instance is running, you can convert it to a reservation to lock in a lower rate for a specified duration.
3. **Update duration** — You can extend or adjust the reservation duration while it's active.

Reservations are automatically unbound when an instance is deactivated or terminated.

## Converting a Running Instance

You can convert any running instance into a reserved one directly from the CloudRift Console. Navigate to your running instance and use the reservation conversion option to select your desired duration and see the savings.

## Managing Reservations

Reservations can be managed through the CloudRift Console, CLI, and REST API. See the [API documentation](https://api.cloudrift.ai/swagger-ui/) for the full list of reservation operations.
