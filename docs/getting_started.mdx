# Getting Started

Fair Compute platform a platform for sharing computational resources.
The platform consists of three main components tailored for different use cases:

1. [Fair CLI](./setup/cli) for accessing the platform.
2. [Fair Desktop](./setup/desktop) for managing your desktop computers and laptops.
3. [Fair Datacenter Service](./setup/datacenter) for managing and leasing datacenter computational resources.
